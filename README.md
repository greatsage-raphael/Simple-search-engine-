## Running
This project requires Node v12 or higher.

1. Open terminal <kbd>Ctrl + Shift + `</kbd>.
2. cd api
3. Run `yarn install` 
4. Run `yarn start`
2. cd client
3. Run `yarn install` 
4. Run `yarn start`
